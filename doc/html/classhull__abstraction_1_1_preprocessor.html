Detailed Description

Class containing several preprocessing procedures.

This class wraps some preprocessing procedures for point cloud data

  • Voxel grid filter
  • Statistical filter
  • Pass-through filter
  • Conditional filter
  • Radius filter
  • Moving least squares method
  • Normal estimation
